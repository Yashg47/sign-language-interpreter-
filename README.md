# sign-language-interpreter-

## 📌 Overview

This project is a Sign Language Interpreter that converts hand gestures into text in real-time.
It helps bridge the communication gap between hearing-impaired individuals and others.

---

## 🚀 Features

* Real-time hand gesture recognition
* Converts gestures into readable text
* Simple and user-friendly system
* Works using webcam input

---

## 🛠️ Tech Stack

* Python
* OpenCV
* NumPy
* TensorFlow / CNN (for gesture classification)

---

## ⚙️ How It Works

1. Capture hand gestures using webcam
2. Process the image using OpenCV
3. Extract features from the hand region
4. Use a trained CNN model to predict the gesture
5. Display the corresponding text output

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/sign-language-interpreter.git
```

2. Install dependencies:

```bash
pip install opencv-python numpy tensorflow
```

3. Run the program:

```bash
python main.py
```
